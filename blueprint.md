# Project Blueprint

## Overview

This project is a simple web application that provides a lottery number generator. It's built using modern, framework-less web technologies (HTML, CSS, and JavaScript).

## Style, Design, and Features

### Initial Version

*   **Lotto Number Generator:**
    *   A user interface with a button to generate 6 random, unique numbers between 1 and 45.
    *   The generated numbers are displayed in a visually appealing format.
    *   The UI is styled with a clean and modern design.

## Current Plan

*   Implement the initial version of the Lotto Number Generator.
